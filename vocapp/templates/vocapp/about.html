What is this website about?

It's a website to use during idle moments to learn the Italian translation of some English terms and expressions.

Why did you build this website?

It is a side project that I created for two main reasons:
The first is to learn new skills and technologies (primarily Django) by developing my first full-stack application from scratch.
As for the second reason, I took the opportunity to build a web application to create something useful for myself: strengthening my English language skills.

What advantages do I have if I create an account?

Creating an account is not mandatory.
By creating one, you will be able to keep track of the learning progress of each expression, decide whether to review previously seen expressions, or discover new ones.
To create an account, only a username and password are required.
No data collection or any type of tracking takes place.
